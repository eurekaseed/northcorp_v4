# Northcorp 架空コーポレートサイト制作

Northcorp はポートフォリオ用に制作した架空企業のコーポレートサイトです。  
企業の信頼性を感じられるシンプルで視認性の高いデザインを目指し、情報構造からデザイン、実装まで一貫して制作しました。

---

## プロジェクト概要
架空企業 Northcorp の事業内容紹介と企業理念を伝えることを目的として構築したコーポレートサイトです。  
ユーザーが必要な情報にすぐアクセスできるよう、簡潔で読みやすい構成と視線誘導を意識したレイアウトを採用しています。

---

## 担当範囲
- 想定ユーザーと掲載情報の整理
- サイトマップ作成、情報構造設計
- デザイン制作（Figma）
- コーディング（HTML / SCSS / JavaScript）
- レスポンシブ対応

---

## デザインのポイント
- 温かみと硬質さを兼ねたクリアなブルーを基調としたミニマルデザインで、企業の信頼性を表現
- 余白を広めに確保し、視認性を高めるレイアウト
- 写真や見出しのコントラストにより自然な視線誘導を実現

---

## コーディングの工夫点
- FLOCSS による分類でメンテナンスしやすいファイル構造を採用
- JavaScript でスムーススクロールやフェード演出を実装
- スマートフォンでの読みやすさを最優先にした調整を実施

---
