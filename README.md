# Northcorp 架空コーポレートサイト制作

Northcorp の架空コーポレートサイト制作プロジェクトです。  
事業内容を分かりやすく伝えつつ、落ち着いた印象と信頼性のあるデザインを目指して制作しました。

---

## プロジェクト概要
企業情報、事業紹介、会社の理念などをシンプルで読みやすくまとめたコーポレートサイトです。  
ユーザーが必要な情報に迷わず辿り着けるよう、構造と導線設計を重視しました。

---

## 担当範囲
- 想定ユーザーと掲載情報の整理
- サイトマップ作成、情報構造設計
- デザイン制作（Figma）
- コーディング（HTML / SCSS / JavaScript）
- レスポンシブ対応

---

## デザインのポイント
- 硬質なブルーを基調としたミニマルデザインで、企業の信頼性を表現
- 余白を広めに確保し、視認性を高めるレイアウト
- 写真や見出しのコントラストにより自然な視線誘導を実現

---

## コーディングの工夫点
- FLOCSS による分類でメンテナンスしやすいファイル構造を採用
- JavaScript でスムーススクロールやフェード演出を実装
- スマートフォンでの読みやすさを最優先にした調整を実施

---
